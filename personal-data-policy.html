Политика обработки персональных данных

1. ОБЩИЕ ПОЛОЖЕНИЯ // SYSTEM_ROOT

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки данных, предпринимаемые командой Fractal-IX (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этой информации.

1.1. Важнейшей целью Оператор считает соблюдение прав и свобод человека при обработке его персональных данных, в том числе защиты прав на неприкосновенность частной жизни и личную тайну.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях (далее — «Пользователи») веб-сайта Fractal-IX.

2. ОСНОВНЫЕ ПОНЯТИЯ // DEFINITIONS

  • Веб-сайт — совокупность графических материалов и кода, обеспечивающих доступность платформы по адресу Fractal-IX.
  • Пользователь — любой посетитель платформы.
  • Персональные данные — любая информация, относящаяся прямо или косвенно к Пользователю.
  • Обработка данных — любое действие (сбор, запись, хранение, удаление) с персональными данными.

3. ДАННЫЕ ДЛЯ ОБРАБОТКИ // DATA_UNITS

Оператор может обрабатывать следующие категории:

  • Фамилия, имя, отчество;
  • Адрес электронной почты (Email);
  • Номер телефона;
  • Обезличенные данные (файлы «cookie») через сервисы статистики (Google Analytics и другие).

4. ЦЕЛИ ОБРАБОТКИ // CORE_PURPOSE

  • Идентификация Пользователя для доступа к LAB-разделам.
  • Установление обратной связи и обработка заявок.
  • Информирование о новых программах пассивного дохода.
  • Улучшение качества работы платформы и аналитика.

5. БЕЗОПАСНОСТЬ И ХРАНЕНИЕ // ENCRYPTION

Оператор обеспечивает сохранность данных и принимает все возможные технические меры для защиты от неправомерного доступа. Персональные данные Пользователя никогда не будут переданы третьим лицам, за исключением случаев исполнения законодательства Великобритании (UK).

Срок обработки данных неограничен. Пользователь может в любой момент отозвать согласие, направив уведомление на:

support@fractal-ix.fit
